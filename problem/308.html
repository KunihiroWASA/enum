P308: Enumeration of all maximum independent set of a graph

Input: A graph $G = (V, E)$.
Output: All maximum independent set of $G$.
Complexity: $O(2^{0.290|V|})$ total time and polynomial space.
Comment:
Reference: [Beigel1999] (Bibtex)
