P377: Enumeration of all topological sortings of a po set

Input: A partial order set $P$.
Output: All topological sortings of $P$.
Complexity: $O(|P|)$ time per solution.
Comment: $|P|$ is the number of objects in $P$.
Reference: [Varol1981]
