Graph / Chordal graph (Bibtex)

P270: Enumeration of all chordal graphs in a graph
Input:
A graph $G$.
Output:
All chordal graphs in $G$.
Complexity:
$O(1)$ delay with $O(n^3)$ space.
Comment:
Using reverse search.
Reference:
[Kiyomi2006] (Bibtex)
P272: Enumeration of all chordal supergraph that contains a given graph
Input:
A graph $G = (V, E)$.
Output:
All chordal supergraphs each of which contain $G$.
Complexity:
$O(|V|^3)$ time for each and $O(|V|^2)$ space.
Comment:
Reference:
[Kiyomi2006a] (Bibtex)
